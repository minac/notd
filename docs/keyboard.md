# Keyboard shortcuts

| Shortcut    | Action                                          |
| ----------- | ----------------------------------------------- |
| `⌘ N`       | New note                                        |
| `⌘ E`       | Toggle Edit / Preview                           |
| `⌘ ,`       | Open Settings                                   |
| `⌘ W`       | Close window (default macOS — not overridden)   |
| `⌘ Q`       | Quit (default macOS — not overridden)           |
| `Tab`       | Insert two spaces in the editor                 |
| `Esc`       | Close the Settings modal when it's open         |

There is **no global hotkey**, no menubar item, and no way to reconfigure these shortcuts — by design.
